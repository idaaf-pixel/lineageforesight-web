Creative Value Assessment

Three question formats, same five topics. Try each and compare.

Version A

Likert Scale

Rate your agreement with five statements about your organisation's creative practices, from "Strongly Disagree" to "Strongly Agree."

Quick to complete Familiar format Quantifiable Risk of neutral bias
Version B

Scenario-Based

Read a realistic workplace scenario and pick the response that best matches how your organisation would actually behave. Each option maps to a maturity level.

Contextual Harder to game Built-in maturity model Takes longer
Version C

Binary + Branching

Start with a Yes/No question, then answer a targeted follow-up based on your response. Two-step format digs deeper into root causes.

Adaptive Reveals root causes Conversational feel More interactive

All three versions cover the same five topics:

  1. Evaluation Priorities — How creative ideas are assessed
  2. Decision-Making Authority — Who approves or rejects creative work
  3. Measuring Success — How creative outcomes are evaluated
  4. Handling Rejection — How "no" is communicated and received
  5. Creative Risk Appetite — Willingness to experiment
Purpose: This is a format comparison test. Try all three versions and consider: Which felt most natural? Which revealed the most honest answers? Which would institutional users actually complete without disengaging?
